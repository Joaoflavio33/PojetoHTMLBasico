Tags básicas para iniciantes em HTML

Heading Tag:

São códigos HTML usados para estruturar o conteúdo de uma página na web, organizando títulos e subtítulos em diferentes níveis hierárquicos. Basicamente, essas tags indicam para o leitor e também para os mecanismos de busca qual é o assunto principal e como ele se desdobra dentro da página. Sendo h1, utilizado para o seu título principal da web, e do h2 para o h6, utilizado para a função de subtítulos da página.

Sendo elas: h1, h2, h3, h4, h5 e h6

Exemplo da tag h1:

H1: "Seu Título Principal"

Exemplo da tag h2:

H2: "Como a Yoga Melhora a Flexibilidade"

Exemplo da tag h3:

H3: "Como você está se sentindo?"

Exemplo da tag h4:

H4: "Júnior acaba de voltar da escola"

Exemplo da tag h5:

H5: "Vejo que você está no caminho certo!"

Exemplo da tag h6:

H6: "Como estão os estudos?"

Tag "P":

A tag "p" no HTML é usada para definir parágrafos de texto em uma página web. Ela organiza o conteúdo textual, tornando-o mais legível e estruturado. Por padrão, navegadores aplicam um pequeno espaçamento antes e depois de cada parágrafo, ajudando na separação visual do texto.

Exemplo da tag P:

P: "Rodolfo foi ao mercado comprar farinha"

Resultado: "Rodolfo foi ao mercado comprar farinha" Já está com a tag "p" aplicada.

Tag Mark:

A tag no HTML é usada para destacar trechos de texto, geralmente para indicar algo importante ou relevante em um contexto específico. O texto dentro da tag é exibido com um fundo amarelo por padrão (como se estivesse "marcado" com um marcador de texto), mas isso pode ser personalizado com CSS.

Exemplo da tag Mark:

Mark: "Este é um exemplo de texto com uma palavra destacada".

Resultado: "Palavra destacada" Já está com a tag "mark" aplicada.

Tag Small:

A tag "small" no HTML é usada para exibir texto em um tamanho menor em comparação ao texto normal ao redor. Ela é geralmente empregada para conteúdo adicional, como notas, avisos legais ou descrições secundárias.

Exemplo da tag small:

Este é o texto principal. Nota adicional ou informação secundária.

Resultado: "Este é o texto principal." (tamanho normal) "Nota adicional ou informação secundária." (tamanho menor) - Já está com a tag "small" aplicada.

Tag i:

A tag "i" no HTML é usada para aplicar um estilo em itálico ao texto. Historicamente, ela era usada exclusivamente para estilização visual, mas atualmente tem um significado semântico adicional, indicando texto que deve ser lido de forma diferenciada ou que tem algum significado especial, como termos técnicos, títulos de obras, ou palavras estrangeiras.

Exemplo da tag i:

Eu adoro o livro Dom Quixote de Miguel de Cervantes.

Resultado: "Dom Quixote" Já está com a tag "i" aplicada .

Tag u:

A tag "u" no HTML é usada para sublinhar texto. Historicamente, ela era empregada puramente para estilização visual, mas em versões modernas do HTML, ganhou significado semântico, indicando texto que precisa de destaque especial ou tem um significado específico, como nomes próprios em textos chineses ou erros ortográficos em documentos.

Exemplo da tag u:

Você cometeu um erro grave nesta frase.

Resultado: "erro grave" Já está com a tag "u" aplicada.

Tag strong:

A tag "strong" no HTML é usada para indicar texto com forte importância semântica. O navegador, por padrão, exibe o texto dentro da tag em negrito, mas o principal propósito dessa tag é transmitir significado, especialmente para leitores de tela e motores de busca.

Exemplo da tag strong:

Por favor, leia as instruções cuidadosamente antes de continuar.

Resultado: "instruções" Já está com a tag "strong" aplicada.

Tag ol:

A tag "ol" no HTML é usada para criar listas ordenadas, ou seja, listas em que os itens têm uma sequência lógica ou numérica. Cada item da lista é definido por uma tag "li" (list item)

Exemplo da tag ol:

  1. Futebol
  2. Camisa 10
  3. Camisa 11

Resultado: "Futebol, camisa 10, camisa 11" Já está com a tag "ol" aplicada

Tag ul:

A tag "ul" no HTML é usada para criar listas não ordenadas, ou seja, listas cujos itens não têm uma sequência ou hierarquia específica. Cada item da lista é definido pela tag "li" (list item).

Exemplo da tag ul:

  • Futebol
  • Camisa 10
  • Camisa 11

Resultado: "Futebol, camisa 10, camisa 11" Já está com a tag "ul" aplicada

Tag a:

A tag "a" no HTML é usada para criar hiperlinks, que permitem a navegação entre páginas da web, seções de uma mesma página ou até o disparo de ações, como o envio de e-mails ou a abertura de arquivos.

Exemplo da tag a:

Visite nosso site

Resultado: "Visite nosso site" Já está com a tag "a" aplicada.

Tag hr:

A tag "hr" no HTML é usada para inserir uma linha horizontal em uma página web. Ela é geralmente empregada para indicar uma mudança de tópico ou uma separação visual entre seções de conteúdo.

Exemplo da tag hr:

Seção 1

Conteúdo da primeira seção.

Seção 2

Conteúdo da segunda seção.

Resultado: "Seção 1 conteúdo da primeira seção, Seção 2 conteúdo da segunda seção" Já está com a tag "hr" aplicada.

Tag sub:

A tag "sub" no HTML é usada para exibir texto em subscrito, ou seja, ligeiramente abaixo da linha de base do texto normal. Ela é comumente utilizada em fórmulas matemáticas, notações químicas, ou em outras situações onde subscritos são necessários.

Exemplo da tag sub:

A fórmula da água é H2O.

Resultado: "H2O" Já está com a tag sub aplicada.

Tag sup:

A tag "sup" no HTML é usada para exibir texto em sobrescrito, ou seja, ligeiramente acima da linha de base do texto normal. É comumente utilizada em contextos científicos, matemáticos e técnicos, como expoentes em fórmulas matemáticas ou notações químicas.

Exemplo da tag sup:

A fórmula da área de um quadrado é A = l2.

Resultado: "12" Já está com a tag sup aplicada

Tag blockquote:

A tag "blockquote" no HTML é usada para marcar citações em bloco, ou seja, trechos de texto que são citações longas de outra fonte. Essa tag é geralmente renderizada com uma indentação para destacar visualmente o conteúdo citado.

Exemplo da tag blockquote:

"O sucesso é ir de fracasso em fracasso sem perder o entusiasmo." - Winston Churchill

Resultado: "O sucesso é ir de fracasso em fracasso sem perder o entusiasmo. - Winston Churchill" Já está com a tag blockquote aplicada, se você prestar atenção, percebe que a frase em si cria uma margem específica ao ser incluída no HTML.

Tag font:

A tag "font" no HTML era usada em versões anteriores (HTML 4) para estilizar o texto, permitindo definir a cor, o tamanho e a família da fonte diretamente no HTML. No entanto, essa tag foi descontinuada (deprecated) no HTML5 porque contraria o princípio de separar o conteúdo da apresentação, que deve ser feito com CSS.

Exemplos da tag font:

  • Texto em vermelho
  • Texto com tamanho maior
  • Texto em Arial

Resultado: "Texto em vermelho, texto com maior tamanho, texto em arial" Já está com a tag "font" aplicada

Tag del:

A tag "del" no HTML é usada para indicar que um trecho de texto foi removido ou excluído de um documento. O navegador, por padrão, exibe o texto dentro da tag "del" com um traço ou linha no meio (chamado de riscado).

Exemplo da tag del:

Este é o preço antigo: R$50,00.

Resultado: "R$50,00" Já está com a tag "del" aplicada.

Tag abbr:

A tag "abbr" no HTML é usada para representar abreviações ou siglas em um documento. Ela permite adicionar uma explicação ou expansão para o texto abreviado usando o atributo title, o que melhora a compreensão e a acessibilidade do conteúdo.

Exemplo da tag abbr:

Estamos localizados nos EUA.

Resultado: "EUA" Já está com a tag "abbr" aplicada. Se você colocar o mouse encima do texto, conseguirá perceber que tem o significado da abreviação.
